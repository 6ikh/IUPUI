This code is a spell checker and word suggestion program using hashing
You should run the program by inputting make run to the terminal and make clean after you have ran the code. 
When you run the program, it will prompt you to enter a word. It will also notify you that you can exit the code with the number 0.
Then you enter any word, if the word is in the dictionary, it will ouput the word and words with the same first letter. If you enter a single character, the levenshteindistance function
will calcilate the distance between the single character and every word in the dictionary. the program will If you entered a word but it may be misspelled, the program will give you options for word to choose from. 
If you enter a word that does not exist, the program returns false. If you enter a non letter, the program returns false.
